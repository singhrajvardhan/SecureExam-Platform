## 🎯 Overview

**SmartExam** is an advanced **AI-powered online exam system** designed to prevent cheating and ensure fairness.

It uses:
- 🎥 Face detection  
- 🧠 AI-based behavior monitoring  
- 🚫 Tab switching detection  

> ⚡ Built by a team of 20 developers (Cyber Security + Computer Science)

---

## ✨ Features

### 🤖 AI Proctoring
- Face tracking & liveness detection ✅  
- Tab switching detection ✅  
- Suspicious activity alerts ✅  

### 📊 Monitoring
- Real-time student tracking ✅  
- Auto proctoring engine 🚧  

### 📝 Exam System
- Auto scoring + negative marking ✅  
- Teacher dashboard ✅  

### 📈 Analytics
- Student reports 🚧

## 🧪 Local Setup

- Node.js

```bash

cd backend
npm install

# frontend
cd ../frontend
npm install

# AI
cd ../ai-proctor
pip install -r requirements.txt
```
